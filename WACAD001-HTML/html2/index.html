A LINGUAGEM HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de HiperTexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados pelos navegadores.

Alguns dos recursos da linguagem HTML vistos até este momento:

Recursos Tag associadas Descrição
Links <a> Criar Links entre páginas
Listas <ul> e <li>
<ol> e <li> 		Criar listas de itens ordenados
Criar listas de itens nao ordenados
